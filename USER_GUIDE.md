## 1️⃣ 应用简介

**NetGeo Courier（网络地理速递）** 是一款专为校园用户设计的网络测速与评估工具。

### 你能用它做什么？

✅ **实时测速**：一键测试当前网络的下载速度、上传速度和延迟  
✅ **位置记录**：自动记录测速时的地理位置（需要授权）  
✅ **历史查看**：查看过往所有测速记录，支持图表展示  
✅ **智能评估**：基于历史数据，生成网络质量报告与优化建议  
✅ **多账户**：支持注册/登录，数据云端同步  

### 适用场景

- 🏫 **教室/图书馆**：测试校园网质量，选择最佳学习位置
- 🏠 **宿舍**：排查网络卡顿原因，判断是否需要报修
- 📅 **长期追踪**：记录每天/每周网速变化，了解高峰时段
- 📊 **对比分析**：对比不同地点的网络表现

## 5️⃣ 核心功能使用说明

### 🚀 功能一：网络测速

#### 操作步骤

1. **进入测速页面**
   - 打开应用后默认为测速页面
   - 或点击底部导航栏的 **"测速"** 按钮

2. **开始测速**
   - 点击页面中央的 **"开始测速"** 大按钮
   - 按钮变为 **"测试中..."**，显示进度条

3. **等待测试完成**
   - 测试过程分为三个阶段（约 30-60 秒）：
     ```
     阶段1：下载测试 → 计算下载速度
     阶段2：上传测试 → 计算上传速度
     阶段3：延迟测试 → 计算 Ping 值
     ```

4. **查看结果**
   - 测试完成后，显示本次测速数据：
     ```
     ⬇️ 下载速度：156.78 Mbps
     ⬆️ 上传速度：45.32 Mbps
     🏓 延迟：24 ms
     ⏰ 时间：2025-06-04 14:30
     ```
   - 数据自动保存到本地并上传到云端

**注意事项**：
- 测速时请保持网络稳定，不要切换 WiFi/移动数据
- 测速结果受当前网络负载影响，建议多次测试取平均值
- 如果测试失败，会提示"测速失败，请重试"，点击重新测试即可

---

### 📍 功能二：位置信息

#### 如何查看位置？

1. 完成测速后，结果卡片上会显示位置信息

#### 位置来源

- **高德地图定位**：基于 GPS + WiFi + 基站定位
- **精度**：室外 10-50 米，室内 50-200 米
- **更新**：每次测速都会重新定位

#### 位置不正确？

- 尝试到窗户附近（GPS 信号更强）
- 检查手机定位服务是否开启
- 在设置中允许应用使用"高精度定位"

---

### 📊 功能三：网络评估报告

#### 什么是网络评估？

应用会根据你的测速记录，计算网络质量综合评分，并给出优化建议。

#### 查看评估报告

1. 点击底部导航栏的 **"评估"** 按钮
2. 页面显示：
   ```
   📈 网络质量评估
   
   综合评分：72 分
   评级：良好 ⭐⭐⭐⭐
   
   历史统计（基于 15 次测速）：
   • 平均下载：148.56 Mbps
   • 平均上传：42.10 Mbps
   • 平均延迟：26 ms
   
   💡 优化建议：
   ✓ 当前网络整体较稳定，建议继续关注校园高峰时段的表现。
   ```

#### 评估维度说明

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| **下载速度** | 平均下载速率 | ≥ 100 Mbps |
| **上传速度** | 平均上传速率 | ≥ 30 Mbps |
| **网络延迟** | 平均 Ping 值 | ≤ 30 ms |
| **综合评分** | 以上指标加权计算 | 85+ 优秀，70-84 良好 |

#### 如何提高评分？

- 📶 **靠近路由器**：信号更强，速度更快
- 🕐 **避开高峰时段**：晚上 8-10 点网络拥堵，建议错峰
- 🔄 **重启路由器**：长时间运行可能导致性能下降
- 📱 **远离干扰源**：微波炉、蓝牙设备会干扰 WiFi 信号
- 🏢 **更换位置**：某些教室/图书馆角落信号覆盖差

### 📋 功能四：历史记录

#### 查看所有测速记录

- **默认排序**：按时间倒序（最新的在最上面）

#### 删除记录

1. 找到要删除的记录
2. 点击右侧的 **🗑️ 删除** 按钮
3. 确认删除

**⚠️ 注意**：删除后无法恢复，且云端同步也会删除。

## 6️⃣ 常见问题

### ❓ 注册时提示"用户名已被注册"

**原因**：该用户名已被其他用户注册。

**解决方法**：
1. 换一个用户名（建议用学号或姓名拼音+数字）
2. 用户名规则：3-64字符，只能包含字母、数字、`_`、`.`、`@`、`-`

---

### ❓ 登录失败，提示"用户名或密码错误"

**可能原因**：
1. 输入错误（大小写、空格）
2. 密码忘记

**解决方法**：
- 检查 Caps Lock 是否开启
- 尝试在电脑上输入（避免手机输入法问题）
- 如果忘记密码，目前需联系管理员重置（后续版本将支持自助重置）

---

### ❓ 测速速度很慢，准确吗？

**测速原理**：应用会从公共测速服务器下载/上传数据块，计算实际吞吐量。

**影响因素**：
- 🏷️ **当前网络负载**：如果很多人同时用网，速度会下降
- 📶 **信号强度**：离路由器越远，信号越弱
- 🔧 **设备性能**：旧手机的网络模块性能可能较差
- 🌐 **服务器位置**：测速服务器在公网上，跨运营商可能限速

**提高准确性**：
1. 关闭其他应用（特别是视频、下载）
2. 多次测试取平均值（建议测 3 次）
3. 在不同时间段测试（早晚对比）
4. 在不同位置测试（教室 vs 图书馆）

---

### ❓ 位置信息不准确或为空

**可能原因**：
1. 未授权位置权限
2. 在室内或信号遮挡严重区域
3. GPS 尚未定位成功

**解决方法**：
1. **授权权限**：设置 → 应用 → NetGeo Courier → 权限 → 位置 → 允许
2. **到开阔地带**：靠近窗户或室外
3. **等待定位**：首次定位可能需要 30-60 秒
4. **检查定位模式**：设置 → 位置 → 选择"高精度模式"

---

### ❓ 数据上传失败/同步失败

**可能原因**：
1. 网络连接断开
2. 后端服务器维护中
3. Token 过期（长时间未使用）

**解决方法**：
1. 检查网络是否正常
2. 退出应用重新登录
3. 如果持续失败，可能是服务器问题，请联系管理员

**离线使用**：
- 测速数据会先保存在本地
- 网络恢复后自动同步
- 查看历史记录不受影响

---

### ❓ 应用闪退/卡死

**解决方法**：
1. **强制停止并重启**：
   - 设置 → 应用 → NetGeo Courier → 强制停止
   - 重新打开应用

2. **清除缓存**（不会删除数据）：
   - 设置 → 应用 → NetGeo Courier → 存储 → 清除缓存

3. **重新安装**：
   - 卸载应用
   - 重新安装最新版本
   - ⚠️ 注意：卸载会清除本地未同步的数据！

---

### ❓ 如何修改个人信息？

**当前版本**：暂不支持修改用户名、显示名。

**未来版本将支持**：
- 修改显示名称
- 修改学号
- 更换密码

---

### ❓ 一个账号可以在多台设备登录吗？

**可以**，但需要注意：
- ✅ 多设备登录是允许的
- ⚠️  Token 会同步到新设备
- ⚠️  如果在一台设备退出登录，其他设备仍然有效（除非 Token 过期）

**安全建议**：
- 不要借给他人使用
- 定期更换密码
- 退出登录时选择"退出当前设备"

---

### ❓ 测速记录保存多久？

**数据保留策略**：
- 云端：永久保存（除非用户删除）
- 本地：缓存最近 100 条记录（再多会自动清理最旧的）

**备份建议**：
- 重要的测速报告可生成 HTML 并分享保存
- 定期查看历史记录，防止误删

---

### ❓ 为什么评估页面显示"暂无数据"？

**原因**：你的测速记录数量不足。

**要求**：
- 至少需要 **1 条** 测速记录才能评估
- 建议上传 **5 条以上** 才能得到有意义的评估

**解决方法**：
1. 完成至少一次测速
2. 等待数据上传到云端（查看历史记录确认已保存）
3. 刷新评估页面

---

### ❓ 如何删除账号？

**当前版本**：暂不支持自助删除。

**联系管理员**：
- 提供你的用户名
- 说明删除原因
- 管理员会在后台删除所有相关数据

**数据删除后**：
- ✅ 所有测速记录被删除
- ✅ 评估结果消失
- ✅ 无法恢复

## 7️⃣ 隐私说明

### 我们收集哪些数据？

| 数据类型 | 用途 | 是否上传云端 |
|----------|------|--------------|
| **用户名** | 账号识别 | ✅ 是 |
| **测速数据**（下载/上传/延迟） | 生成评估报告 | ✅ 是 |
| **位置信息**（经纬度） | 标记测速地点 | ✅ 是 |
| **时间戳** | 记录测速时间 | ✅ 是 |
| **设备信息**（型号、Android 版本） | 统计分析（匿名） | ✅ 是 |

### 我们不会收集

- ❌ 个人真实姓名（除非你在"显示名称"填写）
- ❌ 手机号码
- ❌ 通讯录
- ❌ 短信
- ❌ 相册/照片
- ❌ 录音/录像

### 数据如何使用？

1. **个人使用**：你的数据仅你自己可见（除非你分享报告）
2. **统计分析**：管理员会查看**匿名聚合数据**（不关联个人身份）用于：
   - 了解全校网络状况
   - 识别网络拥堵区域
   - 优化测速算法

### 数据安全

- 🔒 所有传输使用 HTTPS 加密（通过 FRP 隧道）
- 🔐 密码经过 bcrypt 哈希存储
- 🗑️ 删除账号后，数据将从云端彻底清除
- 📅 数据保留期限：账号永久有效，除非主动删除

### 你的权利

根据隐私保护原则，你拥有：
- **知情权**：知道我们收集什么数据（本文档已说明）
- **访问权**：查看自己的所有测速记录（历史页面）
- **删除权**：删除单条记录或整个账号（联系管理员）
- **更正权**：修改显示名称（后续版本支持）
